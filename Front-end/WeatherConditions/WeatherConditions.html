Weather Conditions

Find the bus lines that have a source-station with no raining conditions.


You need to specify the type of station you are interested in (station type) and the source of the information which can be provided either via local datasets or web service (service type). If you choose the option "source" for the station type, the server will return the stations where it is not raining and the bus lines that you can use from these stations. If you choose the option "destination" for the station type, the server will return the stations that are reachable from points where it is not raining and the bus lines that you can use to get to these stations.
